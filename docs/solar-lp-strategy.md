## 2. メッセージング戦略

### 2-1. コアメッセージ

> **「断られた屋根に、載る。」**

**選定理由**:
- ペルソナAの最大ペインポイント「スレート屋根で断られた」を直撃
- 7文字で記憶に残る短さ
- 「載る」が太陽光パネルの物理的設置と、経営戦略に「載る（乗る）」のダブルミーニング
- 競合が絶対に言えない（建設業許可+軽量パネルの組み合わせはLinKだけ）

**サブコピー候補**:
- 「屋根が、収益を生む。」（MASTER_PLANのヘッドライン案と整合）
- 「3.7kgの太陽光が、工場の電気代を半分にする。」

### 2-2. サポートメッセージ（3本柱）

#### 柱1: 軽さの革命 ── 従来の1/5の重量

| 要素 | 内容 |
|------|------|
| **メッセージ** | 「1枚3.7kg。従来パネルの1/5以下。スレート屋根でも設置可能」 |
| **エビデンス** | 従来パネル約20kg/枚 → ペラペラ太陽光3.7kg/枚（TSP公式スペック） |
| **追加根拠** | 厚さ3mm、両面テープ施工、架台不要 → 屋根への荷重は従来の約1/5 |
| **変換効率** | 23.5%（単結晶。多結晶パネル平均17〜19%を大幅に上回る） |
| **視覚表現** | 従来パネルvsペラペラ太陽光の重量比較インフォグラフィック |

#### 柱2: 建設業許可を持つ唯一の太陽光代理店

| 要素 | 内容 |
|------|------|
| **メッセージ** | 「屋根補強から太陽光設置まで、1社で完結」 |
| **エビデンス** | 太陽光設置業者の多くは電気工事業のみ。屋根工事業・建築一式工事業の許可を持つ代理店はLinKのみ（TSP代理店網内） |
| **追加根拠** | 協力会社60社以上のネットワークで耐震補強〜電気工事まで一貫施工 |
| **数字** | 一般的な太陽光+屋根補強の場合、2社手配で工期+30%・コスト+20〜30%増（業界一般的傾向） |
| **視覚表現** | 「他社: 太陽光業者+建設会社の2社手配」vs「LinK: 1社完結」の比較図 |

#### 柱3: 0円で始められる ── PPA or 即時償却

| 要素 | 内容 |
|------|------|
| **メッセージ** | 「初期費用0円のPPA、または即時償却で税制優遇。選べる2つの導入方法」 |
| **エビデンス（PPA）** | 月額電気代50万円以上の工場が対象。電気代最大50%削減 |
| **エビデンス（自社所有）** | 食品工場事例: 投資回収4.5年。中小企業経営強化税制で即時償却可能 |
| **追加根拠** | 25年リニア出力保証（TSP公式）。設備寿命30年以上（太陽光発電協会推計） |
| **視覚表現** | PPA vs 自社所有の比較表 + 投資回収グラフ |

### 2-3. トーン&マナー定義

| 要素 | 定義 | 具体例 |
|------|------|--------|
| **基本トーン** | 堅実な技術パートナー。煽らない、盛らない | 「電気代を最大50%削減できる可能性があります」（断定しない） |
| **敬称** | 相手を立てる丁寧語。ただし堅すぎない | 「オーナー様」「ご担当者様」 |
| **数字の出し方** | 必ず幅+条件付き | 「月額電気代100万円の場合、年間約600万円の削減が見込めます（設置面積・日照条件による）」 |
| **専門用語** | 初出時にカッコ補足 | 「PPA（電力購入契約：初期費用0円で太陽光パネルを設置する仕組み）」 |
| **フォントサイズ感** | 50〜70代が読みやすい大きめ | 本文16px以上、見出し24px以上（スマホ基準） |
| **カラートーン** | 信頼のネイビー + エネルギーのオレンジ/イエロー | LinKブランドカラーに太陽光らしい暖色をアクセント |
| **写真** | 実際の施工現場写真を最優先 | ストック写真は極力避ける（実写は161%以上クリック増: Leadpages調査） |

### 2-4. 禁止表現（NGワード集）

| 分類 | 禁止表現 | 理由 | 代替表現 |
|------|---------|------|---------|
| **過大広告** | 「必ず儲かる」「絶対に元が取れる」 | 景表法違反リスク。高齢経営者は誇大表現に不信感 | 「投資回収4.5年の実績があります（条件による）」 |
| **煽り** | 「今すぐやらないと損」「期間限定」 | B2B高額商材で煽りは逆効果。信頼を損なう | 「2026年度〜目標策定義務化が始まります」（事実ベース） |
| **曖昧** | 「大幅削減」「かなりお得」 | 数字で語れないメッセージは刺さらない | 「電気代最大50%削減（食品工場A社実績）」 |
| **LinK誇張** | 「業界No.1」「トップクラス」 | 根拠なし。建設業許可という事実だけで十分差別化 | 「建設業許可を持つTSP代理店として」 |
| **会社情報誤記** | 「自社施工」「従業員40名」 | マスターデータと不一致（社員2名+協力会社60社以上） | 「協力会社60社以上のネットワーク」 |
| **AI臭い文末** | 「いかがでしたでしょうか」「〜について解説します」 | ユーザーの信頼低下。prohibited-words.md準拠 | 自然な語り口で締める |
| **不正確な法令** | 「設置義務化」 | 正確には「目標策定義務化」（設置自体は義務ではない） | 「2026年度〜太陽光導入目標の策定が義務化」 |

## 3. LPコンバージョン戦略

### 3-1. AIDCA構造設計

LPを「縦スクロール型1ページ」で構成し、AIDCAの各段階をスライド（セクション）に対応させる。

#### A: Attention（注意喚起）── ファーストビュー

| 要素 | 内容 |
|------|------|
| **目的** | 3秒以内に「自分ごと」と認識させる |
| **ヘッドライン** | **「断られた屋根に、載る。」** |
| **サブヘッド** | 「3.7kg超軽量パネル × 建設業許可。スレート屋根でも太陽光が導入できます」 |
| **ビジュアル** | スレート屋根にペラペラ太陽光が設置された実際の工場写真 |
| **数字フック** | 「電気代 最大50%削減 / 初期費用0円プランあり / 25年出力保証」 |
| **CTA** | 「無料で削減試算する」ボタン + 電話番号（タップ可能） |
| **離脱率目標** | 40%以下（B2B LP平均バウンス率: 55〜65%。Databox 2025） |
| **根拠** | B2B LPの中央値バウンス率63.4%（Databox調査 約1,000社）。ターゲット明確なDM流入は平均より低い離脱率が期待できる |

#### I: Interest（興味喚起）── 課題共感 + 法規制

| 要素 | 内容 |
|------|------|
| **目的** | 「このまま放置するとまずい」と認識させる |
| **セクション1: 課題提起** | 「工場の電気代、5年前と比べてどれくらい上がりましたか？」→ 産業用電力料金推移グラフ（資源エネルギー庁データ） |
| **セクション2: 法規制** | 「2026年度〜太陽光導入目標の策定が義務化されます」→ 対象: 年間エネルギー使用量原油換算1,500kl以上の約12,000事業所（経済産業省方針） |
| **セクション3: 従来の壁** | 「スレート屋根だから断られた」「屋根が古くて耐荷重が心配」→ 共感を生むチェックリスト形式 |
| **スクロール到達目標** | 60%以上がこのセクションまで到達 |

#### D: Desire（欲求喚起）── 製品スペック + コスト削減シミュレーション

| 要素 | 内容 |
|------|------|
| **目的** | 「これなら解決できる」と確信させる |
| **セクション4: 製品紹介** | ペラペラ太陽光のスペック比較表（従来パネル vs ペラペラ） |
| **比較データ** | 重量1/5 / 厚さ3mm / 変換効率23.5% / 耐風速60m/s / 曲面120度対応 / 塩害対応 |
| **セクション5: 導入モデル** | PPA（0円設置）vs 自社所有（即時償却）の比較表 |
| **セクション6: コスト削減試算** | インタラクティブ試算ツール（月額電気代入力 → 年間削減額・回収年数を自動計算） |
| **根拠** | シミュレーションツールへの興味: 企業経営者の61.9%が「簡単にできる経済効果シミュレーション」に関心（エネがえる調査2023） |
| **スクロール到達目標** | 45%以上がこのセクションまで到達 |

#### C: Conviction（確信）── 実績 + 信頼要素

| 要素 | 内容 |
|------|------|
| **目的** | 「この会社に頼んで大丈夫」と安心させる |
| **セクション7: 導入事例** | 食品工場A社（投資回収4.5年）/ 物流倉庫B社 等。可能な範囲で写真+数字 |
| **セクション8: なぜLinKか** | 建設業許可 × TSP正規代理店 × 協力会社60社以上。「屋根補強から太陽光まで1社完結」 |
| **セクション9: 保証・アフター** | 25年リニア出力保証 / 施工後の定期点検 / TSPメーカー保証 |
| **セクション10: FAQ** | 「工場を止めずに工事できる？」「屋根が壊れない？」「補助金は使える？」等 5〜7問 |
| **スクロール到達目標** | 35%以上がこのセクションまで到達 |

#### A: Action（行動）── CTA + フォーム

| 要素 | 内容 |
|------|------|
| **目的** | 迷わずアクションを取らせる |
| **セクション11: 最終CTA** | 「まずは無料で、あなたの工場の削減額を試算します」 |
| **フォーム** | 3〜4フィールド（後述） |
| **電話CTA** | 「今すぐ相談: 03-6825-2464」（ペルソナA向け。スマホでタップ発信） |
| **マイクロコピー** | 「所要時間30秒 / 営業電話は一切しません / 試算結果はメールでお届けします」 |
| **フォーム完了率目標** | フォーム開始者の60%以上が送信完了 |

### 3-2. CTA文言の比較検討

| CTA文言 | 心理的ハードル | 期待CVR倍率 | 推奨度 | 根拠 |
|---------|-------------|------------|--------|------|
| **「無料で削減試算する」** | **低** | **1.5〜2.0x** | **推奨** | 「試算」は具体的成果物を示唆。Madison Miles Media調査: 成果物が明確なCTAは曖昧なCTAの2倍以上のCVR |
| 「無料診断を申し込む」 | 中 | 1.2〜1.5x | 次点 | 「診断」は専門性を感じるが「申し込む」が心理的コミット感を高める |
| 「無料相談する」 | 高 | 1.0x（基準） | 非推奨 | Madison Miles Media A/Bテスト結果: 「Free Assessment」は「Let's Connect」にCVRで225%負けた。「相談」は30分の時間コミットを想起させる |
| 「資料をダウンロード」 | 最低 | 1.8〜2.5x | 併用推奨 | CVRは最高だがリード質が低い。ペルソナB向けに「比較資料PDF」として併設 |
| 「見積もりを依頼」 | 最高 | 0.5〜0.8x | 非推奨 | B2B高額商材で初回接点に「見積もり」は時期尚早。ファネルの2段階目で使う |

**最終推奨**: メインCTA = **「無料で削減試算する」** / サブCTA（ペルソナB向け）= **「比較資料をダウンロード」**

### 3-3. フォームフィールド設計

#### メインフォーム（削減試算用）: 4フィールド

| # | フィールド | 入力方式 | 必須 | なぜこの項目か |
|---|-----------|---------|------|--------------|
| 1 | **会社名** | テキスト | 必須 | リード質の担保。法人フィルタリング。営業時の事前調査に必要 |
| 2 | **月額電気代（概算）** | ドロップダウン（50万未満/50〜100万/100〜200万/200万以上） | 必須 | PPA対象かどうかの即時判定（50万円以上が対象）。試算に必須の変数 |
| 3 | **お名前** | テキスト | 必須 | パーソナライズされた試算結果の送付に必要 |
| 4 | **電話番号 or メールアドレス** | テキスト（どちらか一方でOK） | 必須 | ペルソナAは電話派、ペルソナBはメール派。選択肢を与えることで離脱を防ぐ |

**設計根拠**:
- フォームフィールド数3〜5が最適（Unbounce 2025: 5フィールド以下で120%CVR向上）
- 各フィールド追加ごとに20〜30%のCVR低下（Genesys Growth 2026統計）
- 4フィールドは「質の高いリード獲得」と「低い離脱率」のバランス最適点
- 「月額電気代」を聞くことで、営業チームが架電前にPPA対象か判断可能 → 商談効率向上

#### サブフォーム（資料請求用: ペルソナB向け）: 3フィールド

| # | フィールド | 入力方式 | 必須 |
|---|-----------|---------|------|
| 1 | 会社名 | テキスト | 必須 |
| 2 | 部署・役職 | テキスト | 任意 |
| 3 | メールアドレス | テキスト | 必須 |

## 4. KPI目標

### 4-1. LP CVR目標

| 指標 | 業界ベンチマーク | LinK目標（初期） | LinK目標（6ヶ月後） | 根拠 |
|------|----------------|-----------------|-------------------|------|
| **LP CVR（全体）** | 1.8%（太陽光業界: First Page Sage 2026） | **2.5%** | **4.0%** | DM経由のターゲット流入はCold Trafficより高CVR。B2B LPの上位25%は5%超（Unbounce 2025） |
| **LP CVR（DM経由）** | 3〜5%（DM→LP: Postalytics 2025） | **3.5%** | **5.0%** | DM受領者はプレ認知済み。QRコード経由は意図的な訪問 |

## 5. 競合との差別化マトリクス

### 5-1. 太陽光業者LP比較表

| 比較項目 | 一般的な太陽光業者LP | LinK 太陽光LP |
|---------|-------------------|--------------|
| **ターゲット訴求** | 「太陽光でコスト削減」（汎用的） | **「断られた屋根に載る」（スレート屋根オーナー特化）** |
| **建設業許可** | なし（電気工事業のみ） | **建築一式工事業・屋根工事業あり** |
| **屋根補強対応** | 不可（外注 or 非対応） | **自社ネットワークで一括対応** |
| **パネル種類** | 従来型（20kg/枚） | **ペラペラ太陽光（3.7kg/枚）** |
| **スレート屋根対応** | 「対応不可」と断る | **対応可能（軽量+補強で実現）** |
| **折板屋根対応** | 対応可 | **対応可** |
| **耐震補強+太陽光** | 2社以上の手配が必要 | **1社完結** |
| **導入モデル** | PPA or リース | **PPA + 自社所有（即時償却対応）** |
| **CTA** | 「無料見積もり」「お問い合わせ」 | **「無料で削減試算する」（低ハードル+具体的成果物）** |
| **法規制対応の訴求** | あまり触れない | **2026年度義務化を正面から訴求** |
| **フォーム項目数** | 6〜10項目（社名・住所・電話・メール・屋根面積・電気代・希望時期...） | **4項目（会社名・電気代・名前・連絡先）** |

### 5-2. LinKだけが訴求できるポイント（競合障壁の整理）

| # | 独自訴求ポイント | 競合が真似できない理由 | LP上での表現方法 |
|---|---------------|---------------------|----------------|
| 1 | **スレート屋根OK** | 軽量パネル（TSP代理店）× 屋根補強技術（建設業許可）の組み合わせが必要 | ファーストビューの最大訴求ポイント |
| 2 | **耐震補強+太陽光の一括施工** | 建設業許可なしでは屋根補強工事を受注できない | 「1社完結」の比較図で可視化 |
| 3 | **築古工場への対応実績** | 原状回復・リフォームで培った建物診断ノウハウ | 「築30年の工場でも対応可能」+事例 |
| 4 | **協力会社60社以上のネットワーク** | 一般的な太陽光業者は電気工事の協力会社のみ | 「屋根・鉄骨・防水・電気、全ての専門家がいます」 |
