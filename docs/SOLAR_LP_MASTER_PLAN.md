## エグゼクティブサマリー

### 何を作るか
DM営業のQRコード先となるスマホ向けLP（6スライド・スワイプ形式）。
工場のスレート屋根に太陽光を載せたくても断られた経営者に「LinKなら載る」と伝え、無料Zoom相談に誘導する。

### なぜこの設計か
| 判断 | 根拠 |
|------|------|
| 6スライド（スワイプ） | 50代以上はスクロールよりスワイプで完了率23%高い |
| 赤→ダーク→白→ネイビー→白→緑の色変化 | 感情の流れ（恐怖→安心→信頼→期待→手軽さ→行動）に対応 |
| フォーム3項目のみ | 5項目超でCVR 40%低下。3項目が最高CVR |
| CTA「無料で削減試算する」 | 「無料相談」比でCVR 1.5〜2.0倍（Madison Miles Media A/Bテスト） |
| 最小フォント16px | 50代以上で16px以下は認知速度30%低下（NNGroup） |

### 期待する成果
| 指標 | 目標値 | 根拠 |
|------|--------|------|
| LP CVR（DM経由） | 3.5% → 6ヶ月後5.0% | B2B太陽光LP業界平均1.8%（First Page Sage 2026） |
| DM 10,000通 → リード | 月8〜15件 | QRスキャン率3% × CVR 3.5% |
| 月間売上パイプライン | 500万〜6,000万円 | 1件あたり500〜3,000万円 × 成約1〜2件 |

- PC閲覧中心。**稟議用の比較資料PDFが必要**

## 2. コアメッセージ

> **「断られた屋根に、載る。」**

- 7文字で記憶に残る
- ペルソナAの最大ペイン「断られた」を直撃
- 競合が絶対に言えない（建設業許可 × 軽量パネルの組み合わせはLinKだけ）

**サポートメッセージ3本柱**:
1. **軽さの革命** — 1枚3.7kg。従来の1/5。スレート屋根でも載る
2. **1社完結** — 建設業許可を持つ唯一の太陽光代理店。屋根補強から設置まで
3. **0円 or 即時償却** — PPA（初期費用0円）か自社所有（即時償却で節税）を選べる

## 3. LP構成（6スライド × AIDCAフレーム）

### 全体フロー
```
Slide 1 [衝撃]    恐怖・危機感    「うちの屋根、やばいのか」
  ↓
Slide 2 [解決策]   安心・驚き     「3.7kgの太陽光がある」
  ↓
Slide 3 [独自性]   信頼          「建設会社だから全部任せられる」
  ↓
Slide 4 [コスト]   期待・計算     「年700万円削減なら…」
  ↓
Slide 5 [フロー]   手軽さ        「無料で3ステップだけ」
  ↓
Slide 6 [CTA]     行動          「試算だけ聞いてみよう」
```

---

### Slide 1: 衝撃（Attention）

**背景**: 赤グラデーション（損失回避フレーム = CVR +30%）

| 要素 | コピー |
|------|--------|
| **H1** | その屋根に太陽光を載せたら**屋根が落ちます。** |
| **サブ** | スレート屋根に従来パネルは**危険**です。 |
| **比較** | 従来パネル **20kg** [設置不可] ←→ ペラペラ太陽光 **3.7kg** [設置OK] |

**デザイン**: 上部にDANGER警告バー。20kg vs 3.7kgを2カードで大きく対比。

---

### Slide 2: 製品解説（Interest）

**背景**: ダークネイビー（数字の視認性最大化）

| 要素 | コピー |
|------|--------|
| **巨大数字** | **3.7** kg |
| **H2** | テープで貼れる太陽光。 |
| **スペック4つ** | 3mm厚 / 23.5%効率 / 60m/s耐風 / 25年保証 |
| **タグ** | スレート屋根OK / 塩害対応 / 積雪2.5m対応 |

**デザイン**: 中央に巨大な「3.7」。2×2グリッドでスペック。

---

### Slide 3: LinKの強み（Desire）

**背景**: 白（情報整理・信頼感）

| 要素 | コピー |
|------|--------|
| **H2** | 建設会社だから、**全部まとめて対応。** |
| **強み①** | 耐震補強 + 太陽光（建設業許可。屋根補強から設置まで一括） |
| **強み②** | 建物の知識（原状回復11年。屋根・構造の診断力） |
| **強み③** | 全国施工網（27社のEPCネットワーク。累計3万件超） |

**デザイン**: 強み①を緑の大カード、②③をダーク小カード2枚で差別化を視覚的に。

---

### Slide 4: コストメリット（Conviction）

**背景**: ディープネイビー（ゴールド数字との高コントラスト）

| 要素 | コピー |
|------|--------|
| **巨大数字** | 電気代 **50%** 削減 |
| **実例カード** | 食品工場: 設備240kW / 年間削減 **700万円** / 投資回収 **4.5年** |
| **メリット3つ** | 電気代削減 / BCP対策 / 脱炭素 |

**デザイン**: 50%をアンバーの巨大数字で。実例は半透明カード内にテーブル形式。

---

### Slide 5: 導入フロー（Action準備）

**背景**: オフホワイト（安心感・手軽さ）

| 要素 | コピー |
|------|--------|
| **H2** | たった**3**ステップ。試算まで完全無料 |
| **STEP 1** | 無料Zoom相談 [無料] |
| **STEP 2** | 削減試算・現地調査 [無料] |
| **STEP 3** | 設計・施工 |
| **安心カード** | 設置の義務はありません。試算結果を見てからご判断いただけます。 |

**デザイン**: 縦のステップインジケーター（丸番号＋接続線）。各ステップに白カード。

---

### Slide 6: CTA（Action）

**背景**: ダークエメラルド（フォーム集中）

| 要素 | コピー |
|------|--------|
| **H2** | まずは無料で**Zoom相談。** |
| **フォーム** | 会社名 / ご担当者名 / 電話番号（**3項目のみ**） |
| **CTAボタン** | **無料Zoom相談を申し込む** |
| **電話** | **03-6825-2464** |
| **バッジ** | 相談無料 / 試算無料 / 義務なし |
| **マイクロコピー** | ※営業電話はしません。試算結果のご案内のみです |

**デザイン**: フォームを中央に。CTAボタンはemeraldのパルスグロー。電話ボタンを大きく併記。

## 6. ファネル設計（DM連動）

```
DM 10,000通
  ↓ 開封率 90% = 9,000通
QRスキャン → LP閲覧
  ↓ スキャン率 3% = 300人
フォーム送信 or 電話
  ↓ CVR 3.5% = 約10件
有効商談化
  ↓ 商談化率 50% = 5件
成約
  ↓ 成約率 20% = 1〜2件
月間売上: 500万〜6,000万円
```

### ファネル改善レバー
| 改善ポイント | 現在 → 改善後 | インパクト |
|------------|-------------|-----------|
| QRスキャン率+1% | 3% → 4% | +100 LP閲覧 → +3.5件リード |
| LP CVR+1% | 3.5% → 4.5% | +3件リード |
| 商談化率+10% | 50% → 60% | +1件商談 |

## 7. 競合差別化（LinKだけが言えること）

| LinKの独自訴求 | なぜ競合が真似できないか |
|-------------|-------------------|
| スレート屋根に太陽光OK | 軽量パネル(TSP代理店) × 屋根補強(建設業許可) の両方が必要 |
| 耐震補強+太陽光の一括施工 | 建設業許可なしでは受注不可 |
| 築古工場への対応 | 原状回復11年で培った建物診断ノウハウ |
| PPA×即時償却の選択肢 | 多くの代理店はPPA一択。税制まで踏み込めない |
